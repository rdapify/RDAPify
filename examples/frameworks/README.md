# Framework Integration Examples

Examples showing RDAPify integration with popular Node.js frameworks.

## Supported Frameworks

- **Express** - REST API examples
- **Next.js** - API routes and SSR
- **NestJS** - Module and service integration
- **Fastify** - High-performance API
- **Koa** - Middleware examples

## Structure

Each framework has its own subdirectory with:
- Complete working example
- Configuration files
- README with setup instructions

## Running Examples

```bash
cd express-example
npm install
npm start
```

## Common Patterns

- Request/response handling
- Error middleware
- Caching strategies
- Rate limiting
- Authentication

## Contributing

When adding framework examples:
1. Use latest framework version
2. Follow framework best practices
3. Include error handling
4. Add tests
