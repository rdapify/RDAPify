# TypeScript Examples

TypeScript examples demonstrating type-safe RDAPify usage.

## Features

- Full TypeScript type definitions
- Type inference examples
- Generic type usage
- Interface implementations

## Running Examples

```bash
# Using ts-node
npx ts-node domain-lookup.ts

# Or compile first
tsc domain-lookup.ts
node domain-lookup.js
```

## Type Safety Benefits

- Compile-time error checking
- IntelliSense support
- Refactoring safety
- Better documentation

## Examples Include

- Typed domain lookups
- Custom type guards
- Generic query functions
- Error type handling
