# Real RDAP Examples

Examples demonstrating direct RDAP protocol usage and low-level operations.

## Contents

- Raw RDAP queries
- Protocol-level examples
- Bootstrap service usage
- Registry-specific queries
- Response parsing

## Purpose

These examples show:
- How RDAP protocol works
- Direct registry communication
- Response format variations
- Error handling at protocol level

## Use Cases

- Understanding RDAP internals
- Debugging registry issues
- Custom protocol implementations
- Educational purposes

## Running Examples

```bash
node raw-query.js
node bootstrap-discovery.js
node registry-specific.js
```

## Note

For most use cases, use the high-level RDAPify API instead of these low-level examples. These are provided for advanced users and educational purposes.
