# Security Advisories

This directory contains security advisories and vulnerability disclosures for RDAPify.

## Advisory Format

Each advisory includes:
- CVE identifier (if applicable)
- Severity rating (Critical, High, Medium, Low)
- Affected versions
- Description
- Mitigation steps
- Fixed versions

## Naming Convention

Advisories are named: `RDAP-YYYY-NNNN.md`
- YYYY: Year
- NNNN: Sequential number

## Subscribing to Advisories

- Watch this repository
- Subscribe to GitHub Security Advisories
- Follow release notes
