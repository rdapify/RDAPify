# Diagrams

This directory contains Mermaid diagram sources for visualizing RDAPify's architecture and workflows.

## Available Diagrams

- **anomaly_detection.mmd** - Anomaly detection system flow
- **architecture_overview.mmd** - High-level system architecture
- **cache_strategy.mmd** - Caching strategy and invalidation
- **data_flow.mmd** - Data flow through the system
- **discovery_flow.mmd** - RDAP server discovery process
- **error_state_machine.mmd** - Error handling state machine
- **normalization_pipeline.mmd** - Data normalization pipeline

## Viewing Diagrams

These diagrams use Mermaid syntax and can be viewed in:
- GitHub (native support)
- VS Code with Mermaid extension
- Online at [mermaid.live](https://mermaid.live)
- Documentation site (Docusaurus with Mermaid plugin)
