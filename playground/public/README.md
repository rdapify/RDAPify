# Playground Public Assets

Static assets for the RDAPify interactive playground web interface.

## Contents

- HTML files
- CSS stylesheets
- JavaScript files
- Images and icons
- Example data

## Structure

```
public/
├── index.html       # Main playground page
├── css/            # Stylesheets
├── js/             # Client-side JavaScript
├── images/         # Icons and graphics
└── examples/       # Example queries
```

## Features

- Interactive query builder
- Real-time response viewer
- Syntax highlighting
- Error visualization
- Example library

## Development

```bash
# Serve locally
npx serve public/

# Or use the full playground
cd ..
npm run dev
```

## Building

Static assets are served directly. No build step required unless using a bundler.
